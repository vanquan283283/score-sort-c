# Score Sort C Program

This is a simple C program that inputs 5 scores and sorts them in descending order.

## Features

- Input scores
- Sort scores from high to low
- Display sorted scores

## Language

- C

## Example

Input:

```txt
70
50
90
80
60
```

Output:

```txt
1番目の人の点数は90です。
2番目の人の点数は80です。
3番目の人の点数は70です。
4番目の人の点数は60です。
5番目の人の点数は50です。
```
